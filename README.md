# 3D Horizon Dragon

A beta storefront for a 3D toy shop built with Next.js, React Three Fiber, and Tailwind CSS.

## What is included

- Home landing page with an interactive 3D toy preview
- Product catalog with sample inventory
- Product detail page with a model viewer and customization stub
- Customizer lab page for file uploads
- About page describing the beta experience

## Getting started

1. Install dependencies:

```bash
npm install
```

2. Run the development server:

```bash
npm run dev
```

3. Open http://localhost:3000 in your browser.

## Notes

- The 3D toy viewer uses a simple geometry-based preview for the beta.
- Product data is currently static inside `lib/products.ts`.
- Future beta updates can add a backend, cart flow, and file upload preview.
